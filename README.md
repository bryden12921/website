# Olivier Mudagiri Website

Welcome to the official website project for Olivier Mudagiri, a Private Land Notary and real estate professional based in Rwanda. This website serves as a platform to showcase Olivier's services, professional background, and contact information.

## Project Structure

The project is organized as follows:

```
olivier-mudagiri-website
├── src
│   ├── main.tsx          # Entry point of the React application
│   ├── App.tsx           # Main application component with routing
│   ├── pages             # Contains different pages of the website
│   │   ├── Home.tsx      # Homepage with introduction and services
│   │   ├── About.tsx     # Biography and professional background
│   │   ├── Services.tsx  # Details of services offered
│   │   └── Contact.tsx    # Contact information and form
│   ├── components        # Reusable components for the website
│   │   ├── Header.tsx    # Navigation header
│   │   ├── Footer.tsx     # Footer with copyright and links
│   │   ├── ServiceCard.tsx # Card component for displaying services
│   │   └── ContactForm.tsx # Form for user inquiries
│   ├── styles            # CSS styles for the website
│   │   └── globals.css    # Global styles
│   ├── data              # Data files for the project
│   │   └── services.ts    # Service details
│   └── types             # TypeScript types and interfaces
│       └── index.ts      # Type definitions
├── package.json          # NPM configuration file
├── tsconfig.json         # TypeScript configuration file
├── vite.config.ts        # Vite configuration file
└── README.md             # Project documentation
```

## Setup Instructions

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd olivier-mudagiri-website
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the development server:**
   ```
   npm run dev
   ```

4. **Open your browser and navigate to:**
   ```
   http://localhost:3000
   ```

## Usage

- Navigate through the website using the header menu.
- Explore the services offered by Olivier on the Services page.
- Learn more about Olivier's professional background on the About page.
- Contact Olivier directly through the Contact page.
